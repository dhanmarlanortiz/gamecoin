Gamecoin

The Gamecoin is a crypto token created under XRPL. This platform offers a unique utilization of the token. Gamecoin is solely used to run the game thus, making it more valuable as time goes by

Game Mechanics

  • Player may win or lose in the game
  • Use GAMECOIN only in the game
  • Minimum bet is 100 gamecoin to be qualified
  • Lucky winner will get 3x gamecoin based on their bet. Example: 1000 gamecoin bets will be 1000x 3 = 3000 gamecoin
  • Spin or draw will be done everyday at 1200H GMT
  • The number of winners will be posted on twitter @bgamecoin
  • The reward will be sent to your Xumm wallet.

How to play

  1. Select lucky number that you want to bet
  2. Scan the QR code or copy the address in xumm wallet app
  3. Send 100 gamecoin or more that you want to play
  4. Lucky winner will get triple based on their bet
